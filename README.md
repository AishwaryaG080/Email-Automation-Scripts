# Email Automation Scripts

Small Python scripts that send scheduled emails automatically using Gmail SMTP.

## Scripts

### ValentinesDay.py
Sends valentines day emails every 2 hours and a final message at the end.

### VishuBirthday.py
Sends birthday wishes at midnight, morning, and evening.

## Requirements
Python 3.x

## Warning
Do not store real passwords in the script. Use environment variables.
